1. Our Commitment

GlitchRealm is committed to ensuring digital accessibility for all users, including people with disabilities. We strive to meet the Web Content Accessibility Guidelines (WCAG) 2.1 Level AA standards and continuously improve the user experience for everyone.

Accessibility is an ongoing effort, and we welcome feedback to help us achieve our goals.

2. Accessibility Features

GlitchRealm includes the following accessibility features:

Keyboard Navigation

  • Full keyboard navigation support (Tab, Enter, Escape, Arrow keys)
  • Visible focus indicators on interactive elements (implemented via accessibility-keyboard-nav.js)
  • Skip to main content link for screen reader users

Screen Reader Compatibility

  • Semantic HTML5 structure (header, nav, main, footer, article, section)
  • ARIA labels and roles for custom components (modals, dropdowns, interactive games)
  • Descriptive alt text for images and icons
  • Announced notifications for dynamic content updates (game submissions, reviews)

Visual Accessibility

  • High contrast color scheme (cyan/magenta on dark background)
  • Minimum font size of 16px for body text
  • Resizable text (up to 200% zoom without loss of functionality)
  • No flashing or strobing content (reduced motion for glitch effects)

Responsive Design

  • Mobile-friendly layout with touch-friendly targets (minimum 44x44px)
  • Viewport meta tag for proper scaling on mobile devices
  • Responsive breakpoints for tablets, phones, and desktop screens

3. Known Accessibility Issues

We are actively working to address the following known issues:

  • Glitch Text Effects: Animated glitch effects may not be accessible to screen readers (static fallback text provided)
  • Game Content: Individual games submitted by developers may not meet WCAG standards (we encourage developers to follow accessibility best practices; see Developer Agreement)
  • Background Animations: Cyberpunk background animations may be distracting for some users (reduced motion preference supported via CSS prefers-reduced-motion)
  • Color Contrast: Some secondary UI elements (badges, tags) may have contrast ratios below WCAG AA (under review)

We are committed to resolving these issues in future updates.

4. WCAG 2.1 Compliance Status

GlitchRealm's target is WCAG 2.1 Level AA conformance. Current compliance status:

  • Perceivable: Mostly compliant (alt text, contrast, text resizing)
  • Operable: Mostly compliant (keyboard navigation, no keyboard traps)
  • Understandable: Compliant (clear language, predictable navigation, error identification)
  • Robust: Compliant (valid HTML5, ARIA roles, semantic markup)

See the Accessibility & Performance Guide for detailed technical documentation.

5. Assistive Technologies Supported

GlitchRealm is tested with the following assistive technologies:

  • Screen Readers: NVDA (Windows), JAWS (Windows), VoiceOver (macOS/iOS), TalkBack (Android)
  • Keyboard-Only Navigation: Full platform access without a mouse
  • Browser Zoom: Up to 200% zoom without horizontal scrolling (text reflow)
  • High Contrast Modes: Windows High Contrast Mode, browser extensions
  • Speech Input: Dragon NaturallySpeaking, Voice Control (macOS)

6. Third-Party Content

GlitchRealm hosts games submitted by third-party developers. While we encourage accessibility best practices, we cannot guarantee that all user-generated content meets WCAG standards.

If you encounter accessibility barriers in a specific game, please report it to accessibility@glitchrealm.ca, and we will contact the developer.

7. Requesting Accommodations

If you need assistance accessing content or features on GlitchRealm, we are here to help. You may request:

  • Alternative formats for content (text-only version of game descriptions, downloadable guides)
  • Clarification on how to use specific features with assistive technologies
  • Suggestions for accessibility improvements

Contact us at accessibility@glitchrealm.ca with your request. We will respond within 2 business days.

8. Feedback and Reporting Issues

We welcome feedback on accessibility barriers or areas for improvement. To report an accessibility issue:

  1. Email accessibility@glitchrealm.ca with a detailed description
  2. Include the page URL, browser/device information, and assistive technology used (if applicable)
  3. Describe the barrier you encountered and your expected behavior

We will acknowledge your report within 2 business days and provide updates on resolution timelines.

9. Accessibility Roadmap

Planned accessibility improvements include:

  • Q1 2026: Full WCAG 2.1 Level AA audit and remediation
  • Q2 2026: Accessibility settings panel (font size, contrast, reduced motion toggles)
  • Q3 2026: Game developer accessibility guidelines and submission checklist
  • Q4 2026: Screen reader optimization for game launcher and community features

10. Legal Compliance

GlitchRealm strives to comply with accessibility laws and regulations, including:

  • Canada: Accessible Canada Act (ACA)
  • United States: Americans with Disabilities Act (ADA), Section 508
  • Europe: EN 301 549 (European accessibility standard)
