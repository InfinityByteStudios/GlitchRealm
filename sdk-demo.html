SDK Demo

Interactive demonstration of the GlitchRealm Game SDK

Authentication

Sign in to test SDK features that require authentication.

Playtime Tracking

Automatic playtime tracking starts when signed in. Check your current playtime:

const seconds = await sdk.getPlaytime(); console.log(`Total playtime: ${seconds} seconds`);

Achievements

Unlock achievements and track player progress:

await sdk.unlockAchievement('first-win', { name: 'First Victory', description: 'Win your first match', icon: 'trophy.png' });

Event Tracking

Track custom game events for analytics:

await sdk.trackEvent('enemy-defeated', { enemyType: 'boss', location: { x: 100, y: 200 } });

Leaderboards

Submit scores and view rankings:

await sdk.submitScore(1500, { level: 10, difficulty: 'hard' });
